❓ Troubleshooting

API Key Issues

Error: "API Key not recognized"

  • Verify key is from https://console.anthropic.com (not OpenAI)
  • Check for extra spaces when pasting
  • Try regenerating the key in the console

Response Errors

Error: "Timeout" or "Network error"

  • Check your internet connection
  • Verify Claude API status at https://status.anthropic.com
  • Try again in a few seconds

Feature Not Working

Feature returns blank or error

  • Check browser console (F12) for error messages
  • Verify API key is correctly saved
  • Try a simpler demo first (match explanations)
  • Check that sample data is loaded
